🎙️ RaiPod 多模型配置功能

支持配置多个AI模型和语音合成服务，打造个性化播客体验

✨ 新功能特性

🤖 多AI模型支持
• 支持配置多个AI模型（GPT-4、Claude、本地模型等）
• 可选择默认模型进行播客生成
• 每个模型独立配置API地址、密钥和系统提示词
🎵 多语音配置
• 支持配置多种TTS语音服务
• 分别为角色A（主持人）和角色B（嘉宾）选择不同音色
• 支持自定义cURL命令调用任意TTS API
⚙️ 灵活配置管理
• 可视化配置界面，支持增删改查
• 实时预览当前选中的配置
• 配置数据本地存储，安全可靠
🎨 优化用户体验
• 现代化UI设计，操作简单直观
• 支持配置折叠/展开，节省空间
• 智能验证，确保配置完整性

🔧 配置示例

以下是新的配置数据结构示例：

AI模型配置

{ "agentConfigs": [ { "id": "agent_1", "name": "GPT-4 播客专家", "apiUrl": "https://api.openai.com/v1/chat/completions", "apiKey": "sk-...", "model": "gpt-4", "systemPrompt": "你是一个专业的播客主持人..." }, { "id": "agent_2", "name": "Claude 对话助手", "apiUrl": "https://api.anthropic.com/v1/messages", "apiKey": "sk-ant-...", "model": "claude-3-sonnet", "systemPrompt": "你是一个知识渊博的播客嘉宾..." } ] }

语音合成配置

{ "ttsConfigs": [ { "id": "voice_1", "name": "男声 - 磁性", "curlCommand": "curl -X POST https://api.tts.com/v1/speak -H 'Authorization: Bearer xxx' -d '{\"text\":\"{text}\", \"voice\":\"male_deep\"}'" }, { "id": "voice_2", "name": "女声 - 温柔", "curlCommand": "curl -X POST https://api.tts.com/v1/speak -H 'Authorization: Bearer xxx' -d '{\"text\":\"{text}\", \"voice\":\"female_soft\"}'" } ] }

应用设置

{ "settings": { "defaultAgentId": "agent_1", "voiceAConfigId": "voice_1", "voiceBConfigId": "voice_2" } }

🚀 使用方法

  1. 安装RaiPod Chrome扩展
  2. 右键点击扩展图标，选择"选项"
  3. 在AI模型配置部分添加你的模型
  4. 在语音合成配置部分添加TTS服务
  5. 选择默认AI模型和角色A/B的语音
  6. 保存配置，开始使用！
🎙️ 开始体验 RaiPod
